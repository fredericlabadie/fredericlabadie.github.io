// 01 / thesismulti-property setup is a handoff problem

Core argument

Multi-property GA4 setup is as much about the handoff as the installation.

Three properties across two platforms is only useful if the agency can navigate all three views after you leave.

What happened

Three sites, configured and handed to the agency before launch.

A well-known interior design brand was expanding from a single Shopify store to three properties: the existing shop, a new WordPress portfolio, and a new WordPress atelier. GA4 and Search Console needed to be live across all three — consistently configured, verified, and in the hands of the managing agency — before the new sites launched.

The work ran from access granted to walkthrough completed in under two weeks. The one post-launch question — why the agency was only seeing shop data — was resolved the same day it was asked.

Situation

A brand expanding to three sites with no prior multi-property measurement.

The existing Shopify store had GA, but the two new WordPress sites (portfolio and atelier) were net-new properties. The managing agency needed to be able to see traffic and Search Console data across all three after handoff — without analyst support for every question.

Task

Configure three properties consistently and leave the agency able to operate them.

The goal was GA4 across all three platforms with Search Console wired in where applicable, a walkthrough so the agency understood what they were looking at, and a clean handoff before the new sites went live.

// 02 / setupconfigure / verify / walk through / hand off

Build layer

Three properties, one consistent measurement layer.

Shopify and WordPress handle GA4 and Search Console differently. The work was making those differences invisible to the agency.

1GA4 multi-property setup

Configured GA4 across the Shopify store and both new WordPress sites so all three properties reported consistently under one account structure.

2Search Console ownership

Set up and verified Search Console for the WordPress properties, then handed ownership to the managing agency so they could add further users and maintain it independently.

3Agency walkthrough

Ran a GA call with the agency team so they could navigate all three property views, understand the data, and field client questions without needing to refer back.

4Post-launch data question

When the agency asked why they were only seeing shop data, the answer was same-day: the other sites had fractional traffic at launch, and GA4 and UA were in agreement — no tracking issue.

Action

The work ran from access to walkthrough in under two weeks.

I set up GA4 and Search Console across all three properties, handed ownership of the WordPress Search Console accounts to the agency, ran a walkthrough call so they could navigate the account independently, and answered the one post-launch data question same-day. The agency confirmed they had Search Console for all properties except the Shopify shop — which handles its own Search Console separately, as expected.

// 03 / resulthandoff complete / agency operational

Result

The handoff was the deliverable.

A setup is only complete when the agency can answer their own questions. The same-day response to the post-launch data question is the signal that it worked.

Outcome

Three properties live, agency independent, no recurring issues.

  • Three GA4 properties configured across Shopify and two WordPress sites, consistent under one account structure.
  • Search Console set up and ownership transferred to the agency for all WordPress properties before launch.
  • Agency walkthrough completed so the team could navigate all three views and explain the data to the client.
  • Post-launch data question resolved same-day: low traffic on new sites is not a tracking failure; GA4 and UA were in agreement.
  • No recurring support required — the agency operated the setup independently after handoff.

Design lesson

A same-day answer to a post-launch question is a setup signal, not a support task.

When the agency's first post-launch question is "why do I only see shop data?" and you can answer it in one email within hours, it confirms the setup was clean and the walkthrough was thorough enough that they knew to ask rather than guess. That is what a complete handoff looks like.
